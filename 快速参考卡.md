# 雪花密语 v1.3 快速参考卡

## 🎯 核心功能速查

### 创建心语
```
1. 点击「凝结心语」
2. 输入内容
3. 选择精华（极光/星尘）
4. [可选] 启用密码保护
5. 点击「凝结心语」
```

### 查看心语
```
普通心语：直接点击查看
加密心语：点击 → 输入密码 → 查看
```

### 画廊管理
```
🔄 刷新：重新加载
🔍 搜索：输入关键词
⬆️⬇️ 排序：切换新旧
🗑️ 删除：永久销毁
```

---

## 🔐 加密速查

### 加密规格
```
算法：AES-256-GCM
密钥：PBKDF2 (100k iterations)
哈希：SHA-256
密钥长度：256 bits
```

### 密码要求
```
最短：6 位
推荐：8+ 位
包含：字母 + 数字 + 符号
```

### 安全提示
```
✅ 使用强密码
✅ 不重复使用
✅ 及时销毁
⚠️ 密码丢失无法恢复
⚠️ 清除浏览器数据会丢失
```

```

### 存储限制
```
最大记录：50 条
超出处理：自动删除最旧的
存储位置：浏览器本地
```

```
搜索：只能搜索明文
排序：按时间排序

## 🎯 最佳实践

### 创建心语
```
1. 敏感内容 → 使用密码保护
2. 日常记录 → 无需密码
3. 重要内容 → 导出备份
4. 临时信息 → 60秒后自动销毁
```

### 密码管理
```
1. 使用密码管理器
2. 不要使用生日/姓名
3. 每条重要心语不同密码
4. 密码单独记录
```

### 画廊管理
```
1. 定期清理不需要的
2. 重要的导出备份
3. 使用搜索快速查找
4. 使用排序按时间浏览
```
