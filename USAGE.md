# 📖 使用指南

## 🎯 核心功能

### 1. 创建加密雪花

**步骤：**

1. 在首页点击中央的"晶体种子"
2. 输入你想要加密的文字（例如："Hello World 2026"）
3. 选择水晶精华风格：
   - **Aurora Glow**: 极光般的青蓝色调
   - **Stardust Sparkle**: 星尘般的紫色调
4. 点击 "Crystallize Secret" 按钮

**原理：**
- 系统会基于你的文字生成唯一的 Hash 值
- 使用分形算法创建独一无二的雪花图案
- 每个字符的变化都会产生完全不同的雪花

---

### 2. 观赏雪花倒计时

**特性：**

- 雪花会在屏幕中央缓慢旋转
- 60秒倒计时开始
- 你的文字会显示在雪花中心
- 进度条显示剩余时间

**交互：**

- 点击倒计时区域可以进入"Afterglow"模式
- 倒计时结束后，雪花会融化消失
- 融化效果：透明度降低、模糊增加、尺寸放大

---

### 3. 导出雪花艺术品

**步骤：**

1. 在倒计时页面点击倒计时区域
2. 进入 Afterglow（余晖）模式
3. 选择画布类型：
   - Minimalist Postcard（明信片）
   - Desktop Wallpaper（桌面壁纸）
   - Mobile Lockscreen（手机锁屏）
4. 点击 "Export Artwork" 下载

**格式：**
- SVG 矢量格式
- 可无限放大不失真
- 适合打印和数字使用

---

### 4. 探索雪花博物馆

**功能：**

- 查看其他用户创建的雪花残影
- 浏览已融化的秘密
- 获取创作灵感

**访问方式：**

在首页左下角点击 "The Museum"

## 🎨 设计理念

### 为什么是雪花？

1. **独特性**: 自然界中没有两片完全相同的雪花
2. **易逝性**: 雪花融化后无法复原，象征信息的临时性
3. **美学性**: 雪花的六角对称结构天然具有艺术美感
4. **浪漫性**: 雪花承载着冬日的诗意和浪漫

### 分形算法

- 基于 Koch 雪花和 Sierpinski 三角形原理
- 6个主分支，每个分支有多层侧枝
- 复杂度由文字长度和内容决定
- 对称性略有变化，模拟自然雪花的不完美

## 🔒 安全性说明

**重要提示：**

这是一个**艺术性加密项目**，主要用于：
- 浪漫表白
- 临时信息传递
- 艺术创作

**不适合用于：**
- 高度敏感信息
- 商业机密
- 法律文件

如需真正的加密安全，请使用专业加密工具（如 PGP、Signal 等）。

## 🐛 常见问题

**Q: 雪花生成需要多久？**
A: 通常在1-2秒内完成，复杂文字可能需要稍长时间。

**Q: 可以暂停倒计时吗？**
A: 当前版本不支持，这是为了保持"易逝性"的设计理念。

**Q: 导出的图片可以商用吗？**
A: 可以，你生成的雪花完全属于你。

**Q: 同样的文字会生成同样的雪花吗？**
A: 是的！这是设计特性，相同输入产生相同输出。

**Q: 可以恢复已融化的雪花吗？**
A: 如果你记得原文，可以重新生成相同的雪花。
