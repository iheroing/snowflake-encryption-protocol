# 如何加载预设心语

## 🎯 问题

画廊里看不到预设心语？这是因为你已经有自己创建的心语了，预设不会自动覆盖你的数据。

---

## ✨ 解决方案

我添加了一个**"加载预设"按钮**，你可以手动加载 25 条精美预设！

---

## 📱 使用方法

### 方式 1：画廊空状态（如果你删除了所有心语）

1. 进入「心语画廊」
2. 看到空状态页面
3. 点击 **「加载预设」** 按钮
4. 立即看到 25 条精美心语！

### 方式 2：底部工具栏（推荐）

1. 进入「心语画廊」
2. 看到底部的浮动工具栏
3. 找到 ✨ 图标（第4个按钮）
4. 点击 **「加载预设心语」** 按钮
5. 预设会添加到你现有的心语中！

## 🚀 快速体验

### 步骤 1：启动应用
```bash
npm run dev
```

### 步骤 2：进入画廊
- 打开浏览器
- 点击「心语画廊」

### 步骤 3：加载预设
- 点击底部工具栏的 ✨ 按钮
- 或者点击空状态的「加载预设」按钮

### 步骤 4：查看效果
- 看到 25 条精美心语
- 点击查看雪花效果
- 享受美丽的预设！

## 💡 特性说明

### 智能检测
- ✅ 不会重复添加预设
- ✅ 如果已经有预设，会提示"Presets already loaded"
- ✅ 预设会添加到现有心语的前面

### 预设内容
- 🌙 浪漫诗意：5 条
- 🧠 哲理思考：4 条

- 🎭 文艺清新：3 条
- 💪 励志温暖：2 条
- 🗝️ 神秘密语：3 条
- 📅 纪念日期：2 条

### 时间分布
- 每条预设相隔 1 小时
- 从几小时前开始
- 看起来像是自然创建的

## ❓ 常见问题

### Q: 点击按钮后没反应？
A: 检查浏览器控制台，可能已经加载过了。会显示 "Presets already loaded"

### Q: 预设会覆盖我的心语吗？
A: 不会！预设会添加到你现有心语的前面，不会删除任何数据。

### Q: 可以删除预设吗？
A: 可以！每条预设都可以单独删除，就像删除普通心语一样。

### Q: 如何重新加载预设？
A: 如果你删除了所有预设，可以再次点击"加载预设"按钮。

### Q: 预设的 ID 是什么格式？
A: 预设的 ID 以 `preset_` 开头，例如 `preset_0_1234567890`
