## 📝 接下来要做的

### 第一步：创建 GitHub 仓库

**时间**: 2分钟

1. 访问 https://github.com/new
2. 填写信息：
   - Repository name: `snowflake-encryption-protocol`
   - Description: `❄️ 雪花加密协议 - 基于分形算法的艺术加密应用`
   - Public ✅
3. 点击 "Create repository"
4. 复制仓库地址

### 第二步：推送代码

**时间**: 1分钟

**方式 A - 使用脚本（推荐）**:
```bash
./deploy.sh
```

**方式 B - 手动命令**:
```bash
# 替换 YOUR_USERNAME 为你的 GitHub 用户名
git remote add origin https://github.com/YOUR_USERNAME/snowflake-encryption-protocol.git
git push -u origin main
```

### 第三步：部署到 Vercel

**时间**: 3分钟

**方式 A - 通过网站（最简单）**:
1. 访问 https://vercel.com
2. 使用 GitHub 登录
3. 点击 "Add New..." → "Project"
4. 选择 `snowflake-encryption-protocol`
5. 点击 "Deploy"

**方式 B - 通过 CLI**:
```bash
npm install -g vercel
vercel login
vercel --prod
```
