## 💡 使用技巧

### 技巧 1：按风格浏览
```
1. 使用搜索功能
2. 输入关键词（如"星星"、"雪花"）
3. 找到相关风格的预设
```

### 技巧 2：按时间浏览
```
1. 点击排序按钮
2. 切换到「最旧优先」
3. 从第一条预设开始浏览
```

### 技巧 3：删除不喜欢的
```
1. 找到不喜欢的预设
2. 点击右上角删除按钮
3. 确认删除
```

### 技巧 4：模仿创作
```
1. 找到喜欢的预设风格
2. 点击「创建雪花」
3. 模仿该风格创作
```
