# 雪花密语 v1.5 - 阅后即焚

## 🔥 核心更新

### 真正的"阅后即焚"体验

**简化设计理念**：
- 移除后端依赖，保持纯前端实现
- 移除密码保护功能（简化交互）
- 移除画廊保存（真正的临时消息）
- 强化时间限制和融化效果

---

## ✨ 新增功能

### 1. 自定义时间限制

用户可以选择雪花的存活时间：
- **30秒** - 极速阅后即焚
- **60秒** - 经典一分钟（默认）
- **5分钟** - 短暂留存
- **10分钟** - 中等时长
- **30分钟** - 较长时间

### 2. 强化视觉警告

**创建页面**：
- 红色警告框，明确提示"阅后即焚"
- 说明不会保存到画廊
- 强调时间到期后永久消失

**查看页面**：
- 顶部红色警告条
- 实时倒计时显示
- 进度条动画
- 融化动画效果

---

## 🗑️ 移除功能

### 移除密码保护
- 简化用户流程
- 专注于"临时性"而非"安全性"
- 减少交互步骤

### 移除画廊保存
- 不再自动保存到本地
- 真正的"阅后即焚"
- 用户可以手动截图保存

### 移除后端依赖
- 保持纯前端实现
- 无需服务器和数据库
- 部署更简单

## 🔄 用户流程

### 完整体验流程

```
1. 首页
   ├─ 点击"凝结心语"
   └─ 进入创建页面

2. 创建页面
   ├─ 输入心语
   ├─ 选择精华类型（极光/星尘）
   ├─ 选择时间限制（30秒-30分钟）
   ├─ 看到"阅后即焚"警告
   └─ 点击"凝结心语"

3. 查看页面
   ├─ 看到美丽的雪花图案
   ├─ 看到原始文字
   ├─ 看到倒计时（实时更新）
   ├─ 看到红色警告条
   ├─ 可以截图保存
   ├─ 可以分享
   └─ 时间到期 → 融化动画 → 自动关闭

4. 返回首页
   └─ 可以创建新的雪花
```

## 💡 设计理念

### 为什么简化？

**1. 专注核心体验**
- "阅后即焚"是核心概念
- 临时性 > 安全性
- 美感 > 功能复杂度

**2. 降低使用门槛**
- 无需设置密码
- 无需后端配置
- 无需数据库

**3. 保持纯粹**
- 雪花本身就是短暂的
- 不应该被永久保存
- 珍惜当下的美好

## 🚀 技术实现

### 前端实现
```typescript
// 创建页面
const [ttl, setTtl] = useState(60); // 时间限制

// 查看页面
const [timeLeft, setTimeLeft] = useState(ttl);

useEffect(() => {
  if (timeLeft <= 0) {
    setIsMelting(true); // 融化动画
    setTimeout(() => onClose(), 2000); // 2秒后关闭
    return;
  }
  const timer = setInterval(() => {
    setTimeLeft(prev => prev - 1);
  }, 1000);
  return () => clearInterval(timer);
}, [timeLeft]);
```

### 无需后端
- 纯前端实现
- 无需 API
- 无需数据库
- 无需环境变量
