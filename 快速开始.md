# ❄️ 雪花加密协议 - 快速开始

## 🚀 三步启动

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 启动开发服务器
```bash
npm run dev
```

### 3️⃣ 打开浏览器
访问显示的地址（通常是 http://localhost:3000 或 3001）

---

## ✨ 创建你的第一片雪花

1. **点击晶体种子** - 屏幕中央的发光球体
2. **输入你的秘密** - 例如："Hello World 2026" 或 "在星空下相遇"
3. **选择水晶精华** - 极光辉光 或 星尘闪耀
4. **点击结晶秘密** - 观看独特的雪花诞生！

---

## 🎯 核心功能

### 独特性 ❄️
每个文字都会生成完全不同的雪花图案，基于分形算法和Hash值。

### 易逝性 ⏱️
60秒倒计时后，雪花融化消失，信息永远销毁。

### 艺术性 🎨
生成的雪花极致美丽，可以导出保存作为艺术品。

### 浪漫性 💕
完美的情话传递方式，充满诗意和神秘感。

## 🎨 界面说明

### 首页
- **晶体种子**: 点击进入加密页面
- **雪花博物馆**: 左下角入口，查看示例作品

### 加密页面
- **文字输入框**: 输入你的秘密（支持中英文）
- **水晶精华**: 选择雪花的颜色风格
- **结晶秘密**: 生成独特的雪花

### 解密页面
- **雪花展示**: 你的秘密被编码在旋转的雪花中
- **倒计时**: 60秒后雪花融化
- **捕捉余晖**: 点击进入导出页面

### 余晖页面
- **画布选择**: 选择导出格式（明信片/壁纸/锁屏）
- **导出艺术品**: 下载高清SVG文件

## 💡 小技巧

### 相同文字 = 相同雪花
这是设计特性！如果你记得原文，可以随时重新生成相同的雪花。

### 截图留念
在雪花融化前，可以截图保存这个美丽的瞬间。

### 中文支持
完美支持中文输入，中文字符也会生成独特的雪花图案。

### 长文本
支持较长的文本输入，但建议保持在100字以内以获得最佳视觉效果。

## 🐛 常见问题

**Q: 端口被占用怎么办？**
```bash
npm run dev -- --port 3002
```

**Q: 依赖安装失败？**
```bash
rm -rf node_modules package-lock.json
npm install
```

**Q: 雪花生成需要多久？**
A: 通常在1-2秒内完成，复杂文字可能需要稍长时间。

**Q: 可以暂停倒计时吗？**
A: 当前版本不支持，这是为了保持"易逝性"的设计理念。

**Q: 导出的图片可以商用吗？**
A: 可以，你生成的雪花完全属于你。

**Q: 同样的文字会生成同样的雪花吗？**
A: 是的！这是设计特性，相同输入产生相同输出。

## 🚀 部署到线上

### 最简单的方式 - Vercel

1. 将代码推送到 GitHub
2. 访问 [vercel.com](https://vercel.com)
3. 导入你的仓库
4. 点击 Deploy

**3分钟内上线！** 完全免费，自带全球CDN。

### 其他选择

- **Netlify**: 同样简单，拖拽 `dist` 文件夹即可
- **GitHub Pages**: 免费，适合个人项目
- **自己的服务器**: 上传 `dist` 文件夹

详细部署指南见 [DEPLOYMENT.md](DEPLOYMENT.md)
