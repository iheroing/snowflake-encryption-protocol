## 🐛 常见问题

### Q: 如何更新网站？
```bash
git add .
git commit -m "更新说明"
git push
# Vercel 会自动部署
```

### Q: 如何查看部署日志？
访问 Vercel 仪表板 → 选择项目 → Deployments

### Q: 如何回滚版本？
Vercel 仪表板 → Deployments → 选择历史版本 → Promote to Production

### Q: 如何添加自定义域名？
Vercel 仪表板 → Settings → Domains → Add

## 💡 快速技巧

### 截图最佳时机
- 雪花旋转到最美角度
- 倒计时还有充足时间
- 文字清晰可见

### 推荐文字长度
- 短句: 10-20字（最佳）
- 中句: 20-50字（良好）
- 长句: 50-100字（可用）

### 分享建议
- 快速分享 → 截图
