## 📝 第一步：创建 GitHub 仓库（2分钟）

### 操作步骤

1. **打开浏览器**
   - 访问: https://github.com/new

2. **填写信息**
   ```
   Repository name: snowflake-encryption-protocol
   Description: ❄️ 雪花加密协议 - 基于分形算法的艺术加密应用
   Public ✅
   不要勾选 "Add a README file"
   ```

3. **创建仓库**
   - 点击绿色按钮 "Create repository"

4. **复制仓库地址**
   - 页面会显示类似这样的地址:
   ```
   https://github.com/YOUR_USERNAME/snowflake-encryption-protocol.git
   ```
   - 复制这个地址

---

## 📤 第二步：推送代码（1分钟）

### 在终端运行以下命令

```bash
# 1. 添加远程仓库（替换成你复制的地址）
git remote add origin https://github.com/YOUR_USERNAME/snowflake-encryption-protocol.git

# 2. 推送代码
git push -u origin main
```

### 或者使用一键脚本

```bash
./deploy.sh
```

脚本会引导你完成所有步骤！

---

## 🌐 第三步：部署到 Vercel（3分钟）

### 最简单的方式

1. **访问 Vercel**
   - 打开: https://vercel.com
   - 点击 "Sign Up" 或 "Log In"
   - 选择 "Continue with GitHub"

2. **导入项目**
   - 点击 "Add New..." 按钮
   - 选择 "Project"
   - 在列表中找到 `snowflake-encryption-protocol`
   - 点击 "Import"

3. **部署**
   - Vercel 会自动检测配置
   - 直接点击 "Deploy" 按钮
   - 等待 2-3 分钟

4. **完成！**
   - 部署完成后会显示链接
   - 点击链接访问你的网站
   - 🎉 恭喜！你的应用已上线！

## 🐛 遇到问题？

### 推送失败

```bash
# 检查远程仓库
git remote -v

# 如果需要，删除并重新添加
git remote remove origin
git remote add origin YOUR_REPO_URL
```

### Vercel 找不到仓库

1. 确认已推送到 GitHub
2. 刷新 Vercel 页面
3. 检查 GitHub 授权
