## 📊 Vercel 功能

### 已启用的功能

- ✅ **自动部署**: 推送代码自动更新
- ✅ **HTTPS**: 自动配置 SSL 证书
- ✅ **CDN**: 全球加速
- ✅ **预览部署**: Pull Request 自动预览
- ✅ **回滚**: 可以回滚到任何历史版本

### 可选功能

#### 1. Analytics（分析）
- 查看访问量
- 用户地理分布
- 页面性能指标

**启用方式**:
1. 进入项目 Settings
2. 选择 Analytics
3. 点击 Enable

#### 2. 环境变量
如果需要使用 API（如 Gemini）：
1. 进入 Settings → Environment Variables
2. 添加 `GEMINI_API_KEY`
3. 重新部署

#### 3. 自定义域名
1. 进入 Settings → Domains
2. 添加你的域名
3. 配置 DNS
4. 等待验证
