# 画廊按钮快速参考

## 🎛️ 底部浮动工具栏

### 左侧三个功能按钮

```
┌─────────────────────────────────────────┐
│  🔄  🔍  ⬆️  │  ➕ 创建雪花           │
└─────────────────────────────────────────┘
```

#### 1. 🔄 刷新按钮
- **功能**：重新加载画廊中的所有心语
- **使用场景**：
  - 创建新心语后想立即看到
  - 删除心语后刷新列表
  - 感觉数据不同步时
- **操作**：点击一次即可

#### 2. 🔍 搜索按钮
- **功能**：按内容搜索心语
- **使用场景**：
  - 心语太多，想快速找到某条
  - 记得部分内容，想找完整的
- **操作**：
  1. 点击搜索按钮
  2. 底部弹出搜索框
  3. 输入关键词
  4. 实时显示匹配结果
- **注意**：只能搜索明文心语，加密的需要先解密

#### 3. ⬆️⬇️ 排序按钮
- **功能**：切换排序方式
- **两种模式**：
  - ⬇️ 最新优先（默认）：最近创建的在最前面
  - ⬆️ 最旧优先：最早创建的在最前面
- **使用场景**：
  - 想看最早的心语
  - 想按时间顺序回顾
- **操作**：点击切换

## 🎴 心语卡片上的按钮

```
┌─────────────────────────────┐
│  ⏰ 2小时前    🔒 🗑️ 👁️  │
│                             │
│     [雪花图案]              │
│                             │
│  "此刻的心语..."            │
│  📅 01-18 14:30            │
└─────────────────────────────┘
```

### 右上角三个图标

#### 1. 🔒 锁图标（状态指示）
- **含义**：这条心语是加密的
- **不可点击**：只是状态显示
- **查看方式**：点击整个卡片，输入密码

#### 2. 🗑️ 删除按钮
- **功能**：永久销毁这条心语
- **操作流程**：
  1. 点击删除按钮
  2. 弹出确认对话框
  3. 确认后永久删除
- **安全机制**：
  - 内存中多次覆写数据
  - 从存储中永久删除
  - **无法恢复**
- **使用场景**：
  - 不想保留的心语
  - 敏感信息需要销毁
  - 画廊太满需要清理

#### 3. 👁️ 查看图标（状态指示）
- **含义**：可以点击查看详情
- **不可点击**：只是提示
- **查看方式**：点击整个卡片

## 🎯 完整操作流程

### 查看普通心语
1. 进入画廊
2. 点击心语卡片
3. 弹出详情模态框
4. 查看雪花和内容
5. 可以「重新体验」（60秒倒计时）

### 查看加密心语
1. 进入画廊
2. 找到带 🔒 图标的心语
3. 点击卡片
4. 弹出密码输入框
5. 输入密码（支持回车确认）
6. 解密成功后查看详情

### 搜索心语
1. 点击 🔍 搜索按钮
2. 底部弹出搜索框
3. 输入关键词
4. 实时显示匹配结果
5. 点击结果查看详情

### 删除心语
1. 找到要删除的心语
2. 点击右上角 🗑️ 按钮
3. 确认删除
4. 心语永久销毁

## 💡 使用技巧

### 快速查找
- 使用搜索功能：输入关键词快速定位
- 使用排序功能：按时间顺序浏览

### 批量管理
- 定期清理：删除不需要的心语
- 画廊上限：最多保存 50 条，超过会自动删除最旧的

### 安全建议
- 敏感心语：使用密码保护
- 及时销毁：不需要的心语及时删除
- 强密码：至少 8 位，包含字母数字符号

## ❓ 常见问题

### Q: 为什么搜索不到加密的心语？
A: 加密的心语内容是加密存储的，需要先解密才能搜索。你可以：
1. 记住大概的创建时间，用排序功能找
2. 解密后再搜索

### Q: 删除后能恢复吗？
A: 不能。删除是永久的，会在内存中多次覆写数据后删除。这是为了安全性设计的。

### Q: 刷新按钮有什么用？
A: 主要用于同步数据。虽然大部分情况下会自动刷新，但手动刷新可以确保看到最新状态。

### Q: 排序按钮的图标为什么会变？
A: 图标表示当前的排序方式：
- ⬇️ = 最新优先（新的在上面）
- ⬆️ = 最旧优先（旧的在上面）
